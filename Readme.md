# DevKit JS

DevKit JS is a command-line tool that helps JavaScript developers quickly scaffold backend projects with a clean and scalable structure.

## Why DevKit?

Setting up backend projects repeatedly wastes time.  
DevKit provides a simple CLI to generate production-ready boilerplate so developers can focus on building features.

## Features

- CLI-based project generation
- Express.js backend scaffold
- Clean folder structure
- Zero magic, readable code

## Installation

```bash
npm install -g devkit-js
